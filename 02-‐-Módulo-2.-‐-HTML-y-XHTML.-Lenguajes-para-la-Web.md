``` html
<h2>A Description List</h2>

<dl>
  <dt>Término 1</dt>
  <dd>Definición 1</dd>
  <dt>Término 2</dt>
  <dd>Definición 2</dd>
  <dt>Término 3</dt>
  <dd>
    Definición 3
    <ul>
      <li>Elemento 1</li>
      <li>Elemento 2</li>
      <li>
        Elemento 3
        <ol>
          <li>Subelemento 1</li>
          <li>Subelemento 2</li>
          <li>Subelemento 3</li>
        </ol>
      </li>
      <li>Elemento 4</li>
    </ul>
  </dd>
</dl>

</body>
</html>
```
